RAG is a technique that combines the power of large language models with external knowledge retrieval to enhance the quality and accuracy of generated responses. Here's a breakdown of how RAG works with LLMs:

RAG (Retrieval-Augmented Generation) with LLMs

1. User Query

The process starts with a user query, which is the input question or prompt that needs to be answered.

2. Prompt Builder

The prompt builder constructs an effective prompt by combining the user query with additional context or instructions. This helps guide the LLM to generate more relevant and accurate responses.

3. Retriever

The retriever is responsible for fetching relevant information from various sources. In this case, it includes:

  • Vector Database
  • Google Search
  • Knowledge Base
  • Wolfram Alpha
  • SQL Database

4. LLM

The Large Language Model generates the final response based on the constructed prompt and the retrieved information.

Python Implementation using LangChain

Here's a basic implementation of RAG using LangChain:


import os
from langchain import PromptTemplate
from langchain.llms import OpenAI
from langchain.vectorstores import FAISS
from langchain.embeddings import OpenAIEmbeddings
from langchain.retrievers import GoogleSearchAPIWrapper, WolframAlphaAPIWrapper
from langchain.sql_database import SQLDatabase
from langchain.agents import create_sql_agent, Tool, initialize_agent
from langchain.agents.agent_toolkits import SQLDatabaseToolkit
from langchain.memory import ConversationBufferMemory, ConversationSummaryMemory, CombinedMemory
from langchain.tools import DuckDuckGoSearchRun, WikipediaQueryRun
from langchain.utilities import WikipediaAPIWrapper

# Set your API keys
os.environ["OPENAI_API_KEY"] = "your-openai-api-key"
os.environ["GOOGLE_CSE_ID"] = "your-google-cse-id"
os.environ["GOOGLE_API_KEY"] = "your-google-api-key"
os.environ["WOLFRAM_ALPHA_APPID"] = "your-wolfram-alpha-appid"

# Initialize LLM
llm = OpenAI(temperature=0)

# Initialize embeddings
embeddings = OpenAIEmbeddings()

# Vector Database
texts = [
    "New York City is the most populous city in the United States.",
    "Tokyo is the capital and most populous prefecture of Japan."
]
vector_db = FAISS.from_texts(texts, embeddings)

# Google Search
google_search = GoogleSearchAPIWrapper()

# Knowledge Base
knowledge_base = {
    "New York City": "New York City is the most populous city in the United States, with an estimated population of 8.8 million in 2020.",
    "Tokyo": "Tokyo is the capital and most populous prefecture of Japan, with an estimated population of 13.9 million as of 2021."
}

# Wolfram Alpha
wolfram = WolframAlphaAPIWrapper()

# SQL Database
db = SQLDatabase.from_uri("sqlite:///example.db")
sql_toolkit = SQLDatabaseToolkit(db=db)
sql_agent = create_sql_agent(llm=llm, toolkit=sql_toolkit, verbose=True)

# DuckDuckGo Search
duckduckgo_search = DuckDuckGoSearchRun()

# Wikipedia
wikipedia = WikipediaQueryRun(api_wrapper=WikipediaAPIWrapper())

# Prompt Template
prompt_template = PromptTemplate(
    input_variables=["query", "context", "chat_history", "summary"],
    template="""Answer the following question: {query}
Use this additional context: {context}
Chat history: {chat_history}
Summary of conversation: {summary}
"""
)

# Memory
conversation_memory = ConversationBufferMemory(memory_key="chat_history", input_key="query")
summary_memory = ConversationSummaryMemory(llm=llm, input_key="query")
combined_memory = CombinedMemory(memories=[conversation_memory, summary_memory])

# Tools
tools = [
    Tool(
        name="Vector DB Search",
        func=lambda q: vector_db.similarity_search(q),
        description="Useful for when you need to search the vector database for relevant information."
    ),
    Tool(
        name="Google Search",
        func=google_search.run,
        description="Useful for when you need to answer questions about current events."
    ),
    Tool(
        name="Knowledge Base",
        func=lambda q: next((v for k, v in knowledge_base.items() if k.lower() in q.lower()), None),
        description="Useful for when you need specific information about New York City or Tokyo."
    ),
    Tool(
        name="Wolfram Alpha",
        func=wolfram.run,
        description="Useful for when you need to perform calculations or obtain scientific data."
    ),
    Tool(
        name="SQL Database",
        func=sql_agent.run,
        description="Useful for when you need to query a SQL database for information."
    ),
    Tool(
        name="DuckDuckGo Search",
        func=duckduckgo_search.run,
        description="Useful for when you need to answer questions about general information."
    ),
    Tool(
        name="Wikipedia",
        func=wikipedia.run,
        description="Useful for when you need detailed information on a topic. Input should be a search query."
    ),
]

# Initialize Agent
agent = initialize_agent(tools, llm, agent="zero-shot-react-description", memory=combined_memory, verbose=True)

def retrieve_information(query):
    results = []
    
    # Use all tools to gather information
    for tool in tools:
        try:
            result = tool.func(query)
            if isinstance(result, list):  # For vector DB results
                result = " ".join([str(item) for item in result])
            results.append(f"{tool.name} result: {result}")
        except Exception as e:
            results.append(f"{tool.name} error: {str(e)}")
    
    return "\n".join(results)

def process_query(query):
    retrieved_info = retrieve_information(query)
    
    # Update memory
    combined_memory.save_context({"query": query}, {"output": retrieved_info})
    
    chat_history = conversation_memory.load_memory_variables({})["chat_history"]
    summary = summary_memory.load_memory_variables({})["summary"]
    
    prompt = prompt_template.format(
        query=query,
        context=retrieved_info,
        chat_history=chat_history,
        summary=summary
    )
    
    response = agent.run(prompt)
    return response

# Example usage
if __name__ == "__main__":
    queries = [
        "What is the population of New York City, and how does it compare to Tokyo?",
        "What are some famous landmarks in these cities?",
        "How has the population of these cities changed over the last decade?",
    ]
    
    for query in queries:
        print(f"\nQuery: {query}")
        response = process_query(query)
        print(f"Response: {response}\n")
        print("-" * 50)

Explanation of the Code

This code demonstrates a basic implementation of RAG using LangChain. Let's break down each component:

1. User Query

We start with a user query asking about the population of New York City and how it compares to Tokyo.

2. Prompt Builder

We use LangChain's PromptTemplate to create a template that combines the user query with additional context. This helps guide the LLM to generate a more informed response.

3. Retriever

We set up multiple retrievers to gather information from various sources:

  • Vector Database: We use FAISS with OpenAI embeddings to create a simple vector store.
  • Google Search: We use the GoogleSearchRetriever to fetch information from the web.
  • Knowledge Base: We implement a simple dictionary as a knowledge base.
  • Wolfram Alpha: We use the WolframAlphaRetriever to access computational knowledge.
  • SQL Database: We set up an SQLite database and create an SQL agent to query it.

The retrieve_information function combines results from all these sources.

4. LLM

We use OpenAI's language model to generate the final response based on the prompt and retrieved information.

This implementation showcases how RAG can be used to enhance the capabilities of an LLM by incorporating external knowledge sources and structured data retrieval. The combination of different tools and retrievers allows for a more comprehensive and accurate response to user queries.
